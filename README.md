## About

This is the repo for my portfolio website using Next.js and TypeScript. 

## Getting Started

Run the development server in the root of the project:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
